# CancerCompass 🧭

A free, easy-to-use web application that helps cancer patients and caregivers find grants, financial help, and support programs — filtered by their cancer type and state.

## Features
- 🎯 Filtered results by cancer type and state
- 👤 Optional multi-profile support (stored locally on device only)
- 🏛️ National, state, and cancer-type specific resources
- ✅ All resources manually reviewed before listing
- 📱 Works on all devices (Windows, Mac, iPhone, Android)
- 🔐 Private admin page for resource management

## Tech Stack
- React.js
- Tailwind-inspired CSS
- Deployed on Vercel
- No backend — all data stored locally on user's device

## Getting Started

```bash
npm install
npm start
```

## Admin Page
Access the admin page at `/admin` for resource management.
